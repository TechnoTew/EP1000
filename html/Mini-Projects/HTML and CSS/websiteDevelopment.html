Why learn HTML & CSS?

HTML & CSS are the basics when first learning how to create websites. Websites can be used for a variety of purposes from marketing to documentation, which is what this site is used for. Learning HTML & CSS also paves the way for learners to grasp more advanced technologies like React and VueJS to make beautiful and functional websites.

Basic HTML

HTML stands for Hypertext Markup Language and it provides the structure of the page.

Here are some notable things I've learnt about HTML during Digifab

Construction of Website

This website was made using the Bootstrap 4 Framework. The reason why I used this framework was for the familiarity as I used to use Bootstrap to develop websites during my first semester as a DIT student. Bootstrap inherently styles most elements to be more spaced apart and includes friendly helper classes like container-fluid or img-fluid to help with the responsiveness of the website. Moreover, the navigation bar with Bootstrap can be easily done without having to worry too much about custom styling.
